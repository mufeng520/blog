# Card Grid Layout

## Core Strategy
Modular card-based grid system. Content is organized into discrete, self-contained cards. Ideal for content feeds, product listings, and dashboards.

## Spacing & Density
- Card gap: 16-24px
- Edge padding: 16-24px
- Cards per row: 2-4 depending on viewport
- Internal card padding: 16-20px
- Consistent card sizing with variable internal density

## Typography Rules
- Card title: 16-18px semibold
- Card description: 14px regular, 2-3 lines max
- Metadata/tags: 12px with pill/badge styling
- Avoid long paragraphs inside cards

## Layout Structure
- Responsive grid: 2 columns (mobile) to 4 columns (desktop)
- Cards as primary content containers
- Optional header with filters/sorting above grid
- Masonry or uniform grid depending on content type

## AI Prompt Injection
Use modular card-based grid layout. 2-4 cards per row with 16-24px gaps. Self-contained cards with 16-20px internal padding. Card titles at 16-18px semibold. Short descriptions (2-3 lines max) with metadata pills. Responsive grid with optional filter header. Uniform or masonry grid arrangement. Clean card boundaries with subtle shadows or borders.
