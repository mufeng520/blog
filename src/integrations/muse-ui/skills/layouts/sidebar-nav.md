# Sidebar Navigation Layout

## Core Strategy
Persistent sidebar navigation with a main content area. Tree-structured navigation ideal for tools, documentation, admin panels, and multi-level applications.

## Spacing & Density
- Sidebar width: 240-280px fixed
- Sidebar item height: 40-48px
- Sidebar item gap: 4px
- Main content padding: 24-32px
- Collapsible sub-menu indentation: 16-24px

## Typography Rules
- Sidebar category labels: 11-12px uppercase, muted color
- Sidebar nav items: 14px regular
- Active item: bold with accent background
- Main content follows standard hierarchy

## Layout Structure
- Left sidebar: logo + primary nav + secondary nav + user profile
- Main area: header bar + content zone
- Optional right panel for properties/inspector (three-pane)
- Collapsible sidebar on mobile (hamburger menu)
