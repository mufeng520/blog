# Timeline Flow Layout

## Core Strategy
Step-oriented or chronologically ordered layout. Content flows along a timeline axis. Ideal for onboarding, checkout flows, progress tracking, and step-by-step wizards.

## Spacing & Density
- Step/node gap: 32-48px vertically, 24-32px horizontally
- Timeline line thickness: 2-3px
- Node size: 32-40px diameter
- Content per step: 3-6 elements
- Connecting lines between steps clearly visible

## Typography Rules
- Step number/label: 14-16px bold
- Step title: 18-20px semibold
- Step description: 14px regular
- Current step emphasized, past steps muted, future steps ghosted

## Layout Structure
- Vertical timeline: nodes on left, content on right
- Horizontal timeline: nodes across top, content below
- Progress indicator showing completed/current/upcoming steps
- Each step is a self-contained content block

## AI Prompt Injection
Use timeline flow layout with step-oriented structure. Vertical or horizontal timeline axis with 32-48px step gaps. Nodes at 32-40px connected by 2-3px lines. Each step contains 3-6 elements as self-contained blocks. Current step emphasized, completed steps muted, future steps ghosted. Clear progress indicator. Step numbers and titles prominent with descriptions below.
