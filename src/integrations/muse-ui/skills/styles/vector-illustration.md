# Vector Illustration Design System

Flat vector illustration style with clear black outlines and retro soft color palette.

---

## 1. Visual Theme & Atmosphere

- **Brand Personality:** Playful, approachable, nostalgic, and charmingly simple
- **Emotional Description:** Flat vector illustration with no gradients or 3D effects, reminiscent of vintage prints and classic advertising posters
- **Key Characteristics:**
  - Clear, uniform-thickness black outlines on all elements
  - Geometric simplification reducing complex objects to basic shapes
  - Toy model aesthetic that's cute and approachable
  - Panoramic horizontal compositions
  - Depth through layering and overlap, not atmospheric perspective
  - Warm, vintage color palette

---

**Color Philosophy:** Soft, vintage color palette reminiscent of mid-century prints. Colors are flat (no gradients) and bounded by black outlines. Each color is distinct and purposeful. The palette evokes warmth, nostalgia, and approachability.

---

## 3. Typography Rules

**Font Family:**
- **Headlines:** Large, bold retro serif (Playfair Display, Clarendon, or similar)
- **Subtitles:** All-caps sans-serif (Futura, Avenir)
- **Body:** Clean geometric sans-serif (Futura, Avenir, or similar)

**Hierarchy Table:**

| Role | Font | Size | Weight | Line Height | Letter Spacing | Notes |
|------|------|------|--------|-------------|----------------|-------|
| Hero Title | Retro Serif | 48-64px | 700 | 1.1 | 0.01em | Classic advertising poster feel |
| Section Head | Retro Serif | 32-40px | 700 | 1.2 | 0 | Authority and elegance |
| Subtitle | Sans-serif | 14-16px | 600 | 1.2 | 0.1em | All-caps, inside colored blocks |
| Body | Geometric Sans | 16px | 400 | 1.6 | 0 | Consistent weight throughout |
| Caption | Sans-serif | 12-14px | 400 | 1.4 | 0.01em | Metadata, labels |
| Label | Sans-serif | 11-13px | 600 | 1.2 | 0.05em | All-caps, diagram labels |

**Principles:**
- Retro serif for titles conveys authority and elegance
- All-caps sans-serif subtitles in colored blocks for label-like appearance
- Clean geometric sans-serif for readability
- Think classic advertising posters

## 4. Component Stylings

### Buttons
- **Primary:** Solid Coral Red (#E07A5F) background, white text, 8px border-radius, retro serif font
- **Secondary:** Transparent with 3px Deep Charcoal outline, charcoal text, 8px border-radius
- **Action:** Mustard Yellow (#F2CC8F) background, black text, with decorative element

### Cards & Containers
- Soft Cream (#FFF8F0) or Warm White (#FAF7F0) background
- 3px Deep Charcoal (#2D2D2D) outline border
- 8-12px border-radius
- Padding 24-32px
- Subtle paper texture overlay

### Inputs & Forms
- Cream Off-White (#F5F0E6) background
- 2px Deep Charcoal border
- 8px border-radius
- Focus: Mint Green (#81B29A) outline
- Labels: Geometric sans, 14px, all-caps

### Navigation
- Transparent or Cream background
- Geometric sans-serif, 14px, medium weight
- Active: Coral Red underline or background block

### Image Treatment
- All objects have closed black outlines (coloring book style)
- Rounded line endings, avoid sharp corners
- Trees simplified to lollipop or triangle shapes
- Buildings simplified to rectangular blocks with grid windows
- 2.5D perspective (isometric-like but more free-form)
- Depth through layering and overlap

### Distinctive Components
- **Illustrated Card:** Rounded container with illustration, title in retro serif, charcoal outline
- **Feature Block:** Colored rectangular block with icon illustration and text
- **Scene Composition:** Layered illustration with foreground/background elements
- **Decorative Element:** Radiating lines (sunbursts), pill-shaped clouds, dots, stars
- **Character Figure:** Simple geometric figure with minimal facial detail

---

## 5. Layout Principles

**Spacing System:**
- Base unit: 8px
- Section padding: 48-64px vertical
- Content max-width: 1100px centered
- Generous spacing for illustration breathing room

**Grid & Container:**
- 12-column grid for structured layouts
- Panoramic horizontal compositions for illustrations
- Layered compositions with overlapping elements
- Asymmetric, dynamic layouts

**Whitespace Philosophy:**
- Balanced whitespace around illustrations
- Let illustrations breathe
- Clear separation between text and image areas
- Structured but not rigid

**Border Radius Scale:**
- Small (buttons, tags): 8px
- Medium (cards, containers): 8-12px
- Large (feature sections): 12-16px
- Rounded corners preferred over sharp

---

## 6. Depth & Elevation

| Level | Treatment | Use |
|-------|-----------|-----|
| 0 | Flat, outlined | Base illustrations, background elements |
| 1 | Subtle offset | Slightly elevated cards (1px offset shadow) |
| 2 | Layer overlap | Foreground elements overlapping background |
| 3 | Color block | Elevated colored blocks with outline |

**Shadow Philosophy:** No traditional drop shadows. Create depth through:
- Layering and overlap of illustrated elements
- Color block separation
- Outline borders defining edges
- Flat design with strategic layering

---

## 7. Do's and Don'ts

### Do
- Maintain consistent outline thickness throughout
- Use soft, vintage color palette
- Simplify all objects to basic geometric shapes
- Create depth through layering
- Add playful decorative elements (sunbursts, clouds, stars)
- Use rounded line endings and corners
- Keep illustrations flat with clear black outlines

### Don't
- Use gradients or realistic shading
- Create photorealistic elements
- Use thin or varying line weights
- Include complex detailed illustrations
- Add slide numbers, footers, or logos
- Use 3D effects or perspective distortion
- Break the outline consistency

---

## 8. Responsive Behavior

| Breakpoint | Width | Behavior |
|------------|-------|----------|
| Mobile | < 640px | Single column, illustrations at 80% width, headline 32px |
| Tablet | 640-1024px | 2-column layout, illustrations side-by-side, headline 40px |
| Desktop | > 1024px | Full layout, panoramic illustrations, headline 48-64px |

**Touch Targets:** Minimum 48px for interactive elements

**Collapsing Strategy:**
- Panoramic illustrations become contained on mobile
- Side-by-side illustration + text stacks vertically
- Decorative elements reduce in number
- Layered compositions simplify

**Image Behavior:**
- Illustrations scale proportionally
- Outlines maintain consistent visual weight
- Simplified shapes remain recognizable at all sizes
- Decorative elements may hide on very small screens

## 9. Agent Prompt Guide

### Quick Color Reference
- Background: #F5F0E6 (Cream Off-White)
- Primary Text: #1A1A1A (Black)
- Accent: #E07A5F (Coral Red), #81B29A (Mint Green), #F2CC8F (Mustard Yellow)

### Iteration Guide

1. **Simplify shapes:** Reduce every object to basic geometric forms (circles, rectangles, triangles).
2. **Add outlines:** Apply consistent 3px Deep Charcoal outlines to all elements.
3. **Choose flat colors:** Fill shapes with solid vintage palette colors, no gradients.
4. **Layer for depth:** Create foreground, midground, and background layers.
5. **Add decorations:** Include sunbursts, clouds, stars, and other playful elements.
6. **Round corners:** Use rounded line endings and corners, avoid sharp points.
