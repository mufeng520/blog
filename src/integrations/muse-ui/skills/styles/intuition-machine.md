# Intuition Machine

Technical briefing infographic style with aged paper texture and bilingual explanatory text boxes.

---

## 1. Visual Theme & Atmosphere

- **Brand Personality:** Technical, authoritative, academic, and knowledge-driven
- **Emotional Description:** Academic/technical briefing presentation style with vintage technical print aesthetics and modern clarity
- **Key Characteristics:**
  - Clean 2D or isometric technical illustrations
  - Multiple explanatory text boxes with substantive content
  - Split layouts with visuals and text
  - Information-dense but organized with clear visual hierarchy
  - Vintage blueprint aesthetic with modern information design
  - Bilingual callout labels (English + Chinese)

---

## 2. Color Palette & Roles

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| **Primary** | Aged Cream | #F5F0E6 | Primary background |
| **Secondary** | Warm White | #F5F0E1 | Blueprint paper effect, secondary areas |
| **Primary Text** | Dark Maroon | #5D3A3A | Headlines, titles, bracketed main titles |
| **Body Text** | Near Black | #1A1A1A | Text box content, explanations |
| **Accent 1** | Teal | #2F7373 | Primary illustrations, technical diagrams |
| **Accent 2** | Warm Brown | #8B7355 | Secondary elements, annotations |
| **Accent 3** | Maroon | #722F37 | Titles, emphasis, key terms |
| **Outline** | Deep Charcoal | #2D2D2D | Element outlines, borders, diagram lines |
| **Surface** | Cream Tint | #FAF7F0 | Text box backgrounds, callout containers |

**Color Philosophy:** Warm, aged paper tones with teal as the primary technical accent. Maroon provides authority for titles. Deep charcoal outlines give diagrammatic precision. The palette evokes vintage technical prints and blueprints.

---

## 3. Typography Rules

**Font Family:**
- **Headlines:** Bold display font with vintage character (Courier Bold, monospace, or technical serif)
- **Labels:** Clean sans-serif (Inter, Helvetica) for bilingual callouts
- **Body:** Clean geometric sans-serif (Futura, Avenir) for text box content

**Hierarchy Table:**

| Role | Font | Size | Weight | Line Height | Letter Spacing | Notes |
|------|------|------|--------|-------------|----------------|-------|
| Main Title | Bold Display | 36-48px | 700 | 1.1 | 0.05em | ALL CAPS in brackets [TITLE] |
| English Subtitle | Sans-serif | 16-20px | 400 | 1.3 | 0.02em | Below main title, smaller |
| Section Label | Sans-serif | 14-16px | 600 | 1.2 | 0.03em | Bilingual: "ENGLISH 中文" |
| Body Text | Geometric Sans | 13-15px | 400 | 1.5 | 0 | Text box content |
| Callout | Sans-serif | 12-14px | 600 | 1.3 | 0.02em | Pointing to diagram elements |
| Key Quote | Display | 18-22px | 600 | 1.3 | 0.01em | "KEY QUOTE:" box at bottom |

**Principles:**
- Main titles in ALL CAPS within brackets for vintage technical feel
- Bilingual labels format: "ENGLISH TERM 中文翻译"
- High contrast against aged paper background
- Technical, authoritative presence with vintage character
- Consistent weight throughout body text

## 4. Component Stylings

### Buttons
- **Primary:** Teal (#2F7373) background, white text, 4px border-radius, technical sans-serif font
- **Secondary:** Transparent with 2px Deep Charcoal border, charcoal text, 4px border-radius
- **Action:** Maroon (#722F37) background, white text, sharp corners (0px radius)

### Cards & Containers
- Cream Tint (#FAF7F0) or Warm White (#F5F0E1) background
- 1px Deep Charcoal (#2D2D2D) border
- 4-8px border-radius
- Subtle paper texture overlay
- Padding 16-24px

### Inputs & Forms
- Warm White background
- 1px Deep Charcoal border
- 4px border-radius
- Focus: Teal border, subtle glow
- Labels: Bilingual format, sans-serif, 14px

### Navigation
- Aged Cream background, bottom border in Warm Brown
- Sans-serif, 13px, medium weight
- Active: Teal underline or background highlight

### Image Treatment
- Isometric 3D technical illustrations OR flat 2D diagrams
- Clean black outlines on all elements
- Teal as primary illustration color
- Faded thematic background patterns (circuits, gears, flowcharts)

### Distinctive Components
- **Text Box:** Bordered container with substantive content, 3-5 per slide
- **Bilingual Callout:** Label pointing to diagram element: "ENGLISH 中文"
- **Technical Diagram:** Isometric or flat 2D with labeled components
- **Key Quote Box:** Bottom section with core insight highlighted
- **Thematic Pattern:** Faded background pattern related to topic (circuits, gears, etc.)

---

## 5. Layout Principles

**Spacing System:**
- Base unit: 4px (precise, technical)
- Section padding: 32-48px vertical
- Content max-width: 1100px centered
- Tight, information-dense spacing

**Grid & Container:**
- Split layouts: visuals on left/center, text on right/bottom
- Triptych layouts for complex comparisons
- Information-dense but organized
- Clear visual hierarchy through position and size

**Whitespace Philosophy:**
- Minimal whitespace — information-dense approach
- Organized density, not cluttered chaos
- Clear separation between text boxes
- Structured, blueprint-like precision

**Border Radius Scale:**
- Small (text boxes, buttons): 4px
- Medium (cards, containers): 4-8px
- Large (feature sections): 8px
- Sharp corners preferred for technical feel

---

## 6. Depth & Elevation

| Level | Treatment | Use |
|-------|-----------|-----|
| 0 | Flat, textured | Background paper texture |
| 1 | Subtle border | Text boxes, diagram containers |
| 2 | Light shadow (0 1px 4px rgba(0,0,0,0.08)) | Elevated text boxes |
| 3 | Medium shadow (0 2px 8px rgba(0,0,0,0.1)) | Floating callout labels |

**Shadow Philosophy:** Minimal, subtle shadows. The aged paper texture provides most of the visual depth. Outlines and borders do the heavy lifting for separation. Keep it flat and technical like a blueprint.

---

## 7. Do's and Don'ts

### Do
- Include 3-5 text boxes with substantive content from source material
- Use bilingual labels (English + Chinese) for key elements
- Add faded thematic background patterns related to the topic
- Maintain aged paper texture throughout
- Create clear visual hierarchy with split layouts
- Use isometric or flat 2D technical illustrations
- Include a "KEY QUOTE:" box at bottom with core insight

### Don't
- Create photorealistic renders or artistic 3D scenes
- Leave slides without explanatory text content
- Add title blocks or stamps in corners
- Use gradients or glossy effects
- Add slide numbers, footers, or logos
- Create layouts without clear information hierarchy
- Omit bilingual labels if content is bilingual

---

## 8. Responsive Behavior

| Breakpoint | Width | Behavior |
|------------|-------|----------|
| Mobile | < 640px | Single column, text boxes stack, diagrams simplify |
| Tablet | 640-1024px | Split layout maintained, text boxes 2-column |
| Desktop | > 1024px | Full split/triptych layout, all elements visible |

**Touch Targets:** Minimum 44px for interactive elements

**Collapsing Strategy:**
- Split layouts become stacked on mobile
- Text boxes collapse to single column
- Diagrams simplify or become scrollable
- Bilingual labels may show English only on small screens

**Image Behavior:**
- Technical illustrations scale proportionally
- Diagrams may need simplification on small screens
- Background patterns fade further on mobile
- Callout labels reposition to avoid overlap

## 9. Agent Prompt Guide

### Quick Color Reference
- Background: #F5F0E6 (Aged Cream) / #F5F0E1 (Warm White)
- Primary Text: #5D3A3A (Dark Maroon) / #1A1A1A (Near Black)
- Accent: #2F7373 (Teal) / #722F37 (Maroon)
- Outline: #2D2D2D (Deep Charcoal)

4. **Key Quote Section:** "Bottom section with 'KEY QUOTE:' label in Maroon (#722F37), bold. Quote text in 18px geometric sans-serif. Light border above. Subtle paper texture."

5. **Process Flow:** "Horizontal flow diagram with 5 steps. Each step has an isometric icon in Teal, a bilingual label, and a short description. Steps connected by arrows. Aged Cream background with faded gear pattern."

6. **Knowledge Card:** "Cream Tint (#FAF7F0) card with 1px charcoal border. Title in Dark Maroon, 20px bold. Body text in Near Black, 14px. Bilingual terms highlighted in Teal. Subtle paper texture overlay."

### Iteration Guide

1. **Start with structure:** Define the split or triptych layout before adding content.
2. **Place the diagram:** Position the technical illustration as the visual anchor.
3. **Add text boxes:** Create 3-5 text boxes with substantive, informative content.
4. **Label in bilingual:** Add bilingual callout labels pointing to key diagram elements.
5. **Add background pattern:** Include a faded thematic pattern related to the topic.
6. **Finish with quote:** Add a "KEY QUOTE:" box at the bottom with the core insight.
