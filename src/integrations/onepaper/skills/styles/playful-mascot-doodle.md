# Playful Mascot Doodle Style

## Overview
A vibrant, hand-drawn aesthetic featuring a friendly mascot character integrated with bold typography and playful doodle elements. Perfect for casual, energetic brand communications and social media content.

## Visual Characteristics
- **Line Art**: Thick, imperfect hand-drawn black outlines with slight wobble
- **Color Palette**: Bright primary colors (red, yellow, blue) with high saturation
- **Mascot Style**: Cute, approachable character with exaggerated features and expressive eyes
- **Typography**: Bold, rounded sans-serif fonts, often slightly tilted or curved
- **Decorative Elements**: Stars, hearts, squiggles, speech bubbles, arrows
- **Background**: White or very light solid color, minimal texture

## Composition Rules
- Mascot is typically positioned in the foreground, interacting with text
- Text is large and prominent, often with hand-drawn drop shadows
- Decorative elements scattered organically around the main subject
- Generous use of negative space to maintain readability
- Asymmetrical layouts with dynamic energy

## Prompt Template
```
Create a playful mascot doodle illustration featuring {SUBJECT}.

Style: Hand-drawn cartoon with thick black outlines, bright primary colors, and a friendly mascot character.

Mascot: Cute, expressive character with large eyes and a cheerful expression, positioned prominently.

Typography: Bold, rounded, hand-drawn style text saying "{MAIN_TEXT}" with playful drop shadows.

Decorative Elements: Add stars, hearts, squiggles, and small doodles around the composition.

Background: Clean white or very light pastel, keeping focus on the mascot and text.

Mood: Energetic, fun, approachable, youthful.
```

## Variable Substitutions
- `{SUBJECT}`: Main product, character, or theme
- `{MAIN_TEXT}`: Primary headline or message
- `{SECONDARY_TEXT}`: Optional subheading or tagline
- `{ACCENT_SYMBOL}`: Decorative symbol (star, heart, lightning bolt)

## Example Usage
```
SUBJECT: a coffee cup character
MAIN_TEXT: "Morning Boost!"
SECONDARY_TEXT: "Your daily dose of happiness"
ACCENT_SYMBOL: lightning bolt
```

## Aspect Ratio Adaptations
- **1:1 (Square)**: Centered composition, mascot in middle, text wrapping around
- **16:9 (Landscape)**: Mascot on left, large text on right, horizontal flow
- **9:16 (Portrait)**: Mascot at top, stacked text below, vertical energy
- **4:3 (Standard)**: Balanced layout with mascot slightly off-center
