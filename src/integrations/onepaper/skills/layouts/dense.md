# Dense Layout

## Core Strategy
High information density dashboard style. Maximize content per screen. Ideal for data-heavy tools, admin panels, and analytics interfaces.

## Spacing & Density
- Minimum element gap: 12px
- Edge padding: 16px
- Elements per viewport: 15-25
- Compact padding inside components (8-12px)
- Heavy use of micro-borders and background tints for separation

## Typography Rules
- Smaller base font: 13-14px
- Reduced line-height: 1.3-1.4
- Heavy use of labels, badges, and micro-text
- Monospace fonts for numbers and data

## Layout Structure
- Multi-column grid (3-4 columns), nested sub-grids
- Sidebar + main content + right panel three-zone layout
- Overlapping elements and stacked cards acceptable
- Status indicators and metadata densely packed

## AI Prompt Injection
