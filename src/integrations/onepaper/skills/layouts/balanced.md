# Balanced Layout

## Core Strategy
Classic grid-based design with comfortable spacing. Suitable for most content types and general-purpose interfaces.

## Spacing & Density
- Minimum element gap: 24px
- Edge padding: 20px
- Elements per viewport: 6-10
- Use consistent 8px grid base
- Subtle dividers or hairlines allowed for section separation

## Typography Rules
- Clear H1/H2/H3 hierarchy with 1.2-1.5x line-height
- Body text comfortable at 16px base
- Labels and captions at 12-14px

## Layout Structure
- 2-column or 3-column grid for content cards
- Header + content + action area pattern
- Symmetrical balance with occasional asymmetric accents

## AI Prompt Injection
Use a balanced grid layout with comfortable 24px spacing. 6-10 elements per screen arranged in 2-3 columns. Clear typographic hierarchy. Subtle section dividers. Classic UI proportions with 8px base grid. Symmetrical with minor asymmetric accents for visual interest.
