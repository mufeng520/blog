# Hero-Centric Layout

## Core Strategy
Visual-first design dominated by a large hero image or illustration. Text and actions overlay or sit below the hero. Ideal for marketing, landing pages, and brand storytelling.

## Spacing & Density
- Hero area occupies 50-70% of viewport height
- Content below hero has standard 24-32px spacing
- Elements per viewport: 4-7 (excluding hero)
- Generous vertical rhythm below the fold

## Typography Rules
- Large bold headline over or below hero (32-48px)
- Subtitle in lighter weight for contrast
- Body text minimal; let visuals carry the message
- Text may overlay hero with semi-transparent scrim

## Layout Structure
- Full-bleed hero at top
- Single-column content below hero
- CTA prominently placed within or directly below hero
- Supporting content in staggered or asymmetric blocks
