# Sparse Layout

## Core Strategy
Maximize whitespace and negative space. Single focal point design. Each screen presents only one core message or action.

## Spacing & Density
- Minimum element gap: 48px
- Edge padding: 32px
- Elements per viewport: 3-5 maximum
- Use spatial separation; avoid borders and dividers
- Bottom area can be 40%+ blank

## Typography Rules
- Headline occupies its own line with generous whitespace below
- Body text line width: 40-60 characters
- CTA buttons live in isolated zones, never mixed with paragraphs

## Layout Structure
- Single-column dominant, content vertically centered or top-aligned
- Avoid multi-column complex grids
- Asymmetric balance is preferred over strict symmetry

## AI Prompt Injection
Use extreme minimalism. Maximize white space. Only one focal point per screen. Separate elements with generous padding (48px+). No visible borders or dividers. Limit to 3-5 UI elements per viewport. Single column layout, content vertically centered or top-aligned.
