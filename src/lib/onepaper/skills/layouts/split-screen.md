# Split Screen Layout

## Core Strategy
Screen divided into two distinct zones for comparison, form+preview, or image+content pairing. Ideal for login, onboarding comparisons, and dual-pane tools.

## Spacing & Density
- Split ratio: 50/50, 40/60, or 60/40
- Gap between zones: 0px (flush) or 1-2px divider
- Each zone padding: 32-48px
- Elements per zone: 4-8

## Typography Rules
- Zone headlines can differ in scale for hierarchy
- Left zone often more visual, right zone more textual
- Consistent typography system across both zones

## Layout Structure
- Vertical split: left/right (desktop), stacked (mobile)
- Horizontal split: top/bottom
- One zone can be visual/image/illustration
- Other zone can be form/content/actions
- Optional toggle or switch between zones

## AI Prompt Injection

One zone visual/image-focused, other zone content/action-focused. Flush divider or 1-2px separator. Optional toggle between zones. Balanced but differentiated typography per zone.
