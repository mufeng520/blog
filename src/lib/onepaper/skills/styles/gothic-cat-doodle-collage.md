# Gothic Cat Doodle Collage Style

## Overview
A dark, whimsical aesthetic combining gothic Victorian elements with cute cat illustrations and hand-drawn doodles. Features ornate borders, occult symbols, and a playful-yet-mysterious atmosphere. Perfect for alternative fashion, book covers, and mystical-themed content.

## Visual Characteristics
- **Line Art**: Intricate, ornate Victorian-style borders and frames; delicate cat illustrations
- **Color Palette**: Deep purples, blacks, and blood reds with gold accents; occasional pastel highlights
- **Textures**: Aged parchment, velvet, lace patterns, candle wax drips
- **Typography**: Ornate blackletter or decorative serif fonts; some text in delicate script
- **Iconography**: Cats in various poses, crescent moons, stars, potion bottles, ravens, roses
- **Background**: Dark Victorian interior, moonlit garden, or aged spell book pages

## Composition Rules
- Ornate border frames the entire composition
- Central cat illustration as focal point, often surrounded by decorative elements
- Layered collage effect with overlapping elements
- Asymmetrical balance with heavier visual weight on one side
- Mix of detailed illustration and loose doodle elements

## Prompt Template
```
Create a gothic cat doodle collage featuring {SUBJECT}.

Style: Dark Victorian gothic with cute cat illustrations, ornate borders, and mystical doodle elements.

Central Subject: A {SUBJECT} cat in an elegant or playful pose, surrounded by decorative elements.

Typography: Ornate blackletter font for "{MAIN_TEXT}" with gold accents. Delicate script for "{SECONDARY_TEXT}".

Decorative Elements: Add crescent moons, stars, potion bottles, ravens, roses, and occult symbols.

Borders: Intricate Victorian-style ornate frame with gothic architectural details.

Textures: Aged parchment, velvet patterns, lace overlays, and candle wax drip effects.

Color Palette: Deep purples, blacks, blood reds with gold accents and occasional pastel highlights.

Mood: Mysterious, whimsical, darkly romantic, magical, slightly eerie but cute.
```

## Variable Substitutions
- `{SUBJECT}`: Cat description (black cat, siamese, kitten, etc.)
- `{MAIN_TEXT}`: Primary mystical headline
- `{SECONDARY_TEXT}`: Subtitle or spell incantation
- `{ACCENT_SYMBOL}`: Mystical symbol (crescent moon, pentagram, eye)
- `{LOCATION}`: Gothic setting (graveyard, Victorian parlor, moonlit forest)

## Example Usage
```
SUBJECT: a black cat with glowing green eyes
MAIN_TEXT: "Midnight Whispers"
SECONDARY_TEXT: "Spells and purrs under the moon"
ACCENT_SYMBOL: crescent moon
LOCATION: Victorian library with floating candles
```

## Aspect Ratio Adaptations
- **1:1 (Square)**: Circular ornate frame, central cat portrait, tarot card feel
- **16:9 (Landscape)**: Wide scene with cat in environment, horizontal border details
- **9:16 (Portrait)**: Vertical spell book page, stacked elements, phone wallpaper
- **4:3 (Standard): Classic book cover layout with ornate border and centered illustration
